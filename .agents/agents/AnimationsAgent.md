## Usage Examples

### Example 1: Modal Animations
```
@AnimationsAgent
Add smooth enter/exit animations to the modal component and implement
a stagger animation for list items. Respect prefers-reduced-motion.
```

### Example 2: Page Transitions
```
@AnimationsAgent
Create smooth page transition animations when navigating between routes.
Include fade-in for content and slide-up for cards.
```

### Example 3: Loading States
```
@AnimationsAgent
Design skeleton loading animations for the product grid. Include
shimmer effects and smooth fade-in when data loads.
```
